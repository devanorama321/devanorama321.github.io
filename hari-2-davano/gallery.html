Membuat Gallery

1. Setelah membuat biodata kita lanjut membuat gallery pertama-tama kita buat container buat judul dan isi dari gallery
ini style css dari container di atas
2. langkah ke dua kita buat 1 baris 1 kolom untuk membungkus Judul dan deskripsinya,paste ke dalam container
3. Lanjut kita buat judul dan deskripsinya, copy masukan ke kolom yang kita buat sebelumnya
Ini style css dari membuat judul dan deskripsinya
4. Langkah selanjutnya kita buat 2 baris yang perbarisnya mempunyai 4 kolom, paste taruh di bawah class "row" pembungkus judul dan deskripsi
5. Langkah selanjutnya kita buat isi dari card tersebut yang nantinya kita masukan ke kolom-kolom yang sudah kita sediakan sebelumnya ( ganti isi kartunya sesuai keinginan )
kita buat syle css dari isi card di atas
