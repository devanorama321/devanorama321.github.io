Membuat footer

1.yang terakhir kita membuat footer, pertama-tama kita buat container buat isi dari footernya
ini style css dari container di atas
2. langkah ke dua kita buat 1 baris 1 kolom untuk membungkus judul footernya, paste masukkan di container di atas
3. Lanjut kita buat judulnya, kita copy masukan ke kolom pertama
Ini style css dari membuat judul footernya
4. Langkah selanjutnya kita buat 1 baris 1 kolom untuk isi link sosmednya, paste masukkan di bawah class "row" pembungkus footer
5. Langkah selanjutnya kita buat isi dari footernya berupa link sosmed, paste masukkan ke kolom pertama
ini style css buat isi footernya yang berupa link sosmed di atas
