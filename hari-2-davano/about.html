Membuat About

1. Setelah membuat nav dan hero kita lanjut membuat biodata pertama-tama kita buat container buat judul dan isi dari biodatanya
ini style css dari container di atas
2. langkah ke dua kita buat 1 baris 1 kolom untuk membungkus Judul dan deskripsinya,paste masukkan ke container
3. Lanjut kita buat judul dan deskripsinya,paste masukkan ke kolom pertama
Ini style css dari membuat judul dan deskripsinya
4. Langkah selanjutnya kita buat 1 baris yang barisnya mempunyai 2 kolom, kolom pertama berisi gambar kolom kedua berisi isi biodata, pastekan dibawah class "row" pembungkus judul dan deskripsi
5. Langkah selanjutnya kita buat isi dari kolom pertama yaitu berisi gambar, kita copy dan dimasukkan ke kolom pertama
6. lanjut kita buat isi di kolom kedua yang berisi biodata kita, copy dan salin ke kolom kedua
Ini style css dari isi kolom kedua
