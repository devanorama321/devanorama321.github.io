Membuat Nav & Hero

1. Untuk memulai membuat website, seperti biasa kita butuh struktur HTML. Agar HTML kita terintegrasi dengan bootstrap kita bisa tambahkan link cdn dan css bootstrap
2. Lalu kita akan berintegrasi juga dengan Font Awesome biar mempermudah kita mengambil beberapa icon di dalamnya, masukan link cdn ini terlebih dahulu masukkan ke "head" pada html
3. kita buat dulu container untuk membungkus nav dan isi hero nya nanti
Lanjut kita kasih style css buat container pembungkus nav dan isi hero di atas
4.Lanjut setelah sudah terhubung dengan bootstrap kita manfaatkan Elemen navbar dari bootstrap itu sendiri contoh seperti dibawah ini
kita kasih style css nav dari bootstrapnya biar lebih bagus dan menarik
5.Lanjut kita membuat 1 baris 2 kolom untuk gambar user dan isi hero nya, kita copy lalu masukan ke container header
kita kasih style css class pembungkus dan section
6. Lanjut kita buat isi kolom pertama, kita copy masukan ke kolom pertama
kita kasih style css isi dari kolom pertama tersebut
7.lalu kita isi kolom kedua dengan foto user, copy masukkan ke kolom kedua, kalau mau ambil/upload foto yang lain masuk ke nav bagian asset
