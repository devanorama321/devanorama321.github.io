Membuat Bagian Gallery

1. Setelah membuat bagian about, sekarang kita mulai membuat bagian gallery, Buat section gallery dengan class "container" yang berisi judul gallery, dan 2 baris di mana setiap baris memiliki 4 kolom
2. Setiap kolom tambahkan elemen div dengan class "gallery-photo", "title-photo" dan "photo-1" (nomornya sesuai kolom ke berapa)
3. Kita tinggal tambahkan gambar di setiap class "photo-1" sampai "photo-8" dengan menambahkanya di file css, dan beri style juga untuk class "title-photo". Gambar bisa di upload di asset atau mencari di internet (copy dan paste link nya di url)
