Membuat Bagian Footer

1. Setelah membuat bagian file pdf, Terakhir kita buat bagian footer. tapi sebelum itu kita buat lagi icon dengan class "down" tapi icon nya yang fa-up (agar menghadap ke atas)
2. Seperti biasa kita buat elemen footer dengan class "container-fluid" untuk membungkus, lalu di dalamnya tambahkan elemen div dengan class "container" yang berisi 1 baris dan 3 kolom
3. Untuk kolom pertama kita kasih elemen h4 dan isi nama kita sesuai judul website
4. Untuk kolom kedua kita isikan list link untuk menuju ke bagian dari website kita seperti home, about dan yang lainnya
5. Kolom ketiga kita isikan link sosial media kita
6. Tambahkan style untuk setiap elemen dan class yang sudah kita buat di dalam footer
7. Terakhir untuk responsif di layar lebar maupun layar hp kita tambahkan media query di dalam css bawah sendiri seperti berikut
