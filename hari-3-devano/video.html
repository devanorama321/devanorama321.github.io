Membuat Bagian Video

1. Setelah membuat bagian gallery, sekarang membuat bagian video. Pertama kita buat section video dengan class "container" dan tambahkan terlebih dahulu elemen h2 untuk judul dan tambahkan icon awesome
2. Kita akan menambahkan video youtube terlebih dahulu. Tambahkan elemen h3 dengan judul "youtube", buat class "video" sebagai baris yang di dalamnya ada 5 kolom class "card-video".
3. Di setiap "card-video" kita tambahkan iframe youtube dengan cara mengembed ke html supaya video kita tampil di website. Cara mendapatkan link embed kita masuk ke video youtube kita, klik bagikan, pilih link embed. Tak lupa juga tambahkan "card-body" yang bisa di ambil dari bootstrap untuk judul video. Ukuran embed/iframenya 300 x 200
4. Tambahkan style untuk class "video", dan "card-body"
5. Sekarang kita buat yang video tiktok. Seperti sebelumnya buat judul, class "video-tiktok" dan "card-video"
6. Embed juga link video tiktok kita agar tampil di halaman. Cara mendapatkan link embed dengan cara masuk ke video tiktok kita, klik bagikan dan embed link. Masukkan kedalam "card-video" yang sudah di buat sebelumnya
7. Terakhir tambahkan style css untuk class "video-tiktok"
