Membuat Bagian About

1. Setelah membuat header kita lanjutkan untuk membuat bagian about atau biodata, tambahkan section dengan class "container","about", id "about" dan tambahkan 1 baris dan 2 kolom
2. di dalam baris yang berada di kolom 1 kita tambahkan 3 div dengan class "about-img" yang nantinya kita isi gambar di dalam css
3. Tambahkan style css untuk setiap elemen dan class yang sudah kita tambahkan di kolom 1 seperti "about-img","center" dan yang lainnya
4. Sekarang kita fokus ke kolom ke 2, kita tambahkan slide atau biasa dikenal carousel yang bisa di ambil di Bootstrap atau copy berikut. kita buat 2 slide carousel aktif
5. Pada slide pertama kita bisa tambahkan elemen seperti h4 dan p untuk biodata kita, tak lupa juga ada class "line" untuk membuat garis
6. Pada slide kedua kita buat keahlian kita apa saja dengan animasi progress dengan class yang kita tambahkan seperti "loading-bar", "progress" dan lainnya. Untuk mengubah gambar kita bisa ambil di font-awesome
7. Terakhir kita berikan style css untuk semua elemen dan animasi yang kita tambahkan di slide dua carousel tadi
