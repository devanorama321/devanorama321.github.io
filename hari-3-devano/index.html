Membuat Bagian Header & Hero

1. Pertama buat file "index.html", "style.css", folder img, folder file, kemudian isikan file "index.html" dengan struktur html yang sudah ada link css, cdn bootstrap dan cdn font awesome
2. Kita buat headernya menggunakan navbar yang disediakan Bootstrap atau bisa langsung copy di bawah ini
Berikan style css untuk elemen h1, a, class "navbar" dan body website kita
3. Buat section hero mulai dari memberikan id "home", membuat class "container" dan "hero", serta membuat 1 baris/row dan 2 kolom/col di dalam sectionnya
4. Tambahkan elemen-elemen yang di butuhkan di kolom pertama seperti h1, a dan icon. Perlu di perhatikan kita menambahkan class "change-color" dan "typing" untuk animasinya
5. Tambahkan css untuk elemen yang tadi ditambahkan di section hero dan sudah diberikan nama class masing-masing seperti elemen h1, a, class "navbar", "button-contact", dan animasi seperti "change-color", "typing", dan "moveUpDown".
6. Sekarang kita tambahkan elemen-elemen di kolom ke 2 pada section hero yaitu gambar kotak, svg berwarna dan foto kalian (usahakan 400 x 458 px, di hapus background dan upload di asset). Asset gambar bisa di download dan upload disini
7. Tambahkan css lagi untuk kolom ke dua yang sudah di berikan nama class seperti "kotak", "home__img", "icon-down" dan yang lainnya.
