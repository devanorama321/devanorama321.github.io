Membuat Bagian Video Gallery

Buat section baru dengan class "video" dan beri id berupa "#video". Lalu tambahkan div dengan class "container". Tambahkan tag h3 dengan class "section-title". Tambahkan tag p dengan class "section-desc" untuk membuat deskripsi singkat. Dan tambahkan dua div dengan style seperti pada contoh.
Di dalam div yang pertama, tambahkan tag table dengan class "card" yang memiliki satu baris dengan dua kolom.
- Pada kolom pertama, tambahkan iframe dengan lebar 280 dan source berupa link video yang ingin ditampilkan. Sebagai contoh, silakan embed salah satu video dari YouTube.
- Pada kolom kedua, berikan class berupa "right-col" dan tambahkan tag h4 dengan class "card-title" untuk menuliskan judul video. Lalu tambahkan paragraf untuk menuliskan deskripsi video tadi dan sebuah button untuk mengunjungi video tersebut.
Masih pada div pertama, silakan menyalin table sebelumnya dan tempel di bawah tabel pertama.
Selanjutnya, salin kedua tabel pada div pertama dan tempelkan di dalam div kedua.
Saatnya memberi style untuk tampilan video gallery kita.
