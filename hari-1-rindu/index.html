Membuat Bagian Header & Hero

Buat sebuah folder lalu buat file baru bernama "index.html", "style.css, dan sebuah folder bernama "images". Buka file HTML menggunakan Visual Studio Code kemudian tambahkan struktur HTML. Tambahkan link icon, CSS, dan CDN font awesome (untuk menambahkan icon) agar bisa ter integrasi.
Download logo disini kemudian masukkan ke folder "images"
Mari kita buat navigasi bar. Di dalam tag body tambahkan sebuah div dengan nama class "container". Di dalam div tersebut, buat sebuah section dengan class "navbar" kemudian tambahkan beberapa div lagi seperti contoh berikut:
Tag span pada logo bertujuan untuk membedakan warna pada teks "Nama" dan "Logo". Pada tag a, terdapat href yang mengarah ke id sebuah section ketika teks tersebut di klik.
Berikan style CSS pada class-class tadi.
Selanjutnya membuat hero section. Di bawah section navbar, buat sebuah section dengan class "hero". Tambahkan tag h1 lalu tuliskan namamu. Kemudian tambahkan tag h3 dan beri sebuah class bernama "section-title".
Di bawah tag h3, tambahkan tag p untuk menuliskan sebuah deskripsi singkat tentang dirimu. Kemudian buat div baru dan tambahkan tiga buah tag a. Berikan icon dengan menambahkan sebuah class berupa "fa fa-nama icon". Pada bagian href, tuliskan link sosmed mu.
Berikan style untuk section hero. Gambar untuk background bisa diunduh disini kemudian masukkan ke folder "images"
