Membuat Bagian Photo Gallery

Buat section baru dengan class "galery" dan beri id berupa "#gallery". Lalu tambahkan div dengan class "container". Tambahkan tag h3 dengan class "section-title". Tambahkan tag p dengan class "section-desc" untuk membuat deskripsi singkat. Dan tambahkan dua div dengan style seperti pada contoh.
Di dalam div yang pertama, tambahkan tag table dengan class "card" yang memiliki tiga baris yang memiliki satu kolom.
- Pada baris pertama, tambahkan img dengan class "card-image". Unduh sebuah foto, masukkan ke folder image, dan panggil dengan menambahkan statement "/images/nama-file" pada bagian source image. Atau kalian juga bisa menggunakan foto secara online dengan menambahkan url seperti pada contoh.
- Pada baris kedua, beri class "right-col" di kolomnya dan tambahkan tag h4 dengan class "card-title".
- Pada baris keiga, tambahkan span dengan class "card-desc" dan tuliskan deskripsi singkat tentang foto di ats.
Masih pada div pertama, silakan menyalin table sebelumnya dan tempel sebanyak 4 kali di bawah tabel pertama.
Selanjutnya, salin keempat tabel pada div pertama dan tempelkan di dalam div kedua.
Saatnya memperindah tampilan galeri kita.
