Membuat Bagian Footer

Buat section baru dengan class "footer". Lalu, tambahkan sebuah div dan paragraf yang berisikan copyright.
Di dalam div tadi, tambahkan tiga buah anchor yang berisi icon sosial media kita. Berikan class berupa "fa fa-nama icon". Dan cantumkan juga link menuju sosial media kamu.
Terakhir, tambahkan style CSS untuk bagian footer.
