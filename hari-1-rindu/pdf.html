Membuat Bagian PDF

Buat section baru dengan class "pdf" dan beri id berupa "#pdf". Lalu tambahkan div dengan class "container" di dalamnya.
Di dalam container tadi tambahkan h3 dengan class "section-title" untuk membuat judul section. Selanjutnya tambahkan tag p dengan class "section-desc" untuk menambahkan deskripsi singkat mengenai section ini.
Kemudian, tambahkan div baru dengan class "card" dan berikan style berupa "text-align: center;". Di dalam div tadi tambahkan sebuat object dengan data berupa dokumen (.pdf) yang bisa diunduh disini.
Terakhir, tambahkan style CSS untuk section ini.
