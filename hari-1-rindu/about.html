Membuat Bagian Biodata

Buat section baru dengan class "bio" dan beri id berupa "#bio". Lalu tambahkan div dengan class "container". Tambahkan tag h3 dengan class "section-title". Tambahkan tag p dengan class "section-desc" untuk membuat deskripsi singkat. Dan tambahkan tag table.
Di dalam tag table, tambahkan tag tr. Di dalam tr, tambahkan dua tag th dengan class "bioimage" dan "biodata". Pada bioimage tambahkan sebuah img dengan class "profile". Unduh foto profil disini dan pindahkan ke folder "images". Atau kunjungi unsplash.com dan pilih foto profil sesuai keinginanmu. Pada biodata, beri style berupa "width: 50%;".
Di dalam biodata, tambahkan tag h4 untuk membuat judul, tag span untuk membuat deskripsi singkat, dan tag table untuk membuat tabel baru.
Di dalam biotable, buat 6 buah baris dengan menambahkan tag tr. Di masing-masing baris, tambahkan dua kolom dengan menggunakan tag th. Pada kolom-kolom milik baris genap (baris ke-2, ke-4, dan ke-6) beri class berupa "tableitem". Dan pada dua kolom pada baris terakhir, berikan style berupa "padding-bottom: 0px;"
Saatnya mempercantik tampilan biodata kita.
